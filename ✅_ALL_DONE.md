## 🚀 Launch in 5 Steps

### Step 1: Rename Files
```powershell
ren package-nextjs.json package.json
ren tsconfig-nextjs.json tsconfig.json
```

### Step 2: Install
```powershell
npm install
```

### Step 3: Setup Database
```powershell
npx prisma generate
npx prisma db push
```

### Step 4: Configure
```powershell
copy env.nextjs.example .env.local
notepad .env.local
# Add your Supabase credentials
```

### Step 5: Launch!
```powershell
npm run dev
```

**Open:** http://localhost:3000 🎉
